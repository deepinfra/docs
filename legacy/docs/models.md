title: Models
---

DeepInfra hosts a large number of the most popular machine learning models.
You can find [the full list here](/models), conveniently split into 
categories based on their functionality.

We are constantly adding more. DeepInfra is usualy amongst the first to add a
new model once it is available.

Each model also has a dedicated page where you can quickly try it out or
see its API documentation. Or just grab an example.

We also support deploying [custom models](/docs/advanced/custom_llms) on DeepInfra.

If you think there is a model that we should run just let us know at info@deepinfra.com.
We read every email.
